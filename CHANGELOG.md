# Changelog

## [1.2.0] - 2026-03-09

### Added
- **Cluster Autoscaler Integration**: New autoscaling section in UI
  - `enableAutoscaler` toggle to enable/disable autoscaling
  - `autoscalerMinSize` - minimum nodes in pool
  - `autoscalerMaxSize` - maximum nodes in pool
- US region support (US-EAST-VA-1, US-WEST-OR-1) in fallback options
- b3-series flavors (newer generation) to fallback options
- Autoscaler configuration metadata for driver integration

### Changed
- SSH key field now optional (auto-generated by Rancher if not specified)
- Instance name field renamed to "Instance Name Prefix"
- Updated image names to match OVH API format (e.g., "Ubuntu 24.04")

### Fixed
- Region fallback options now include US regions for ovh-us endpoint

## [1.1.0] - 2026-03-08

### Added
- Initial release with Sneederco branding
- Full OVH Cloud integration for Rancher node provisioning
- Dynamic field loading with API integration
- Fallback options for offline/disconnected scenarios
- Comprehensive validation rules
- Section-based UI organization

## [1.0.0] - 2026-03-07

### Added
- Base UI driver structure
- Core provisioning fields
